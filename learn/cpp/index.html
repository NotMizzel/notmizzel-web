Learn C++ | Getting Started

To start, let's install a compiler. Common options are: G++, Microsoft Visual Studio, and MinGW.
to install G++ in Ubuntu, use this command:
sudo apt install g++
This command installs the g++ compiler on Ubuntu. To install MSVC, you can go to the Official download page.
To install G++ on a Windows system, go to the MinGW website and download the latest version for your architecture.
In the installer, select the G++ options.

g++ script.cpp After compiling the code, you can now run the code. On Windows: .\a.exe On Linux: ./a.out You can also use the flag -o to specify the filename when it is compiled. Example: g++ code.cpp -o thisismyapp
